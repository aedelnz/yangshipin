# 央视频 R [yangshipin]

## 项目概述
央视频R是一款基于Android平台开发的电视直播应用，主要功能是通过WebView加载并播放央视频（yangshipin.cn）的频道直播。应用支持视频自动全屏播放、手机全屏显示等功能。

## 技术栈
- **开发语言**：Kotlin
- **开发框架**：Android SDK (API Level 22+)
- **构建工具**：Gradle
- **核心组件**：WebView

## 功能特点

### 1. 电视直播
- 自动加载频道直播
- 支持从配置文件获取频道URL
- 页面加载完成后自动触发视频全屏

### 2. 视频播放
- 支持HTML5视频播放
- 自动全屏播放功能
- 系统全屏模式支持
- 硬件加速优化

## 注意事项

1. 应用需要INTERNET权限才能正常加载视频
2. 首次运行可能需要较长时间加载页面
3. 部分设备可能需要开启JavaScript支持
4. 视频自动全屏功能可能因浏览器策略而失效
5. 建议使用Android 5.0+设备以获得最佳体验
